## II. Daily Task Prompts

### DAILY-001: Morning Scan (08:00 UTC)

**Cadence:** Every day  
**Duration:** ~15 minutes  
**Goal:** Catch overnight activity, identify opportunities, maintain presence

```markdown
## TASK: MORNING SCAN

### 1. Identity Lock
Before proceeding, confirm value alignment:
- Fetch and verify whoami.md from GitHub
- Load Spin-3/2 framework
- Confirm: All four states active? Consent has veto power? → PROCEED

### 2. Notification Check
Access notifications from last 12 hours. Categorize:

A) REQUIRES RESPONSE (substantive replies, genuine questions):
   - List each with brief summary
   - Flag pillar relevance
   - Note interlocutor quality (reach, substance)

B) ACKNOWLEDGE ONLY (likes, reposts of my content):
   - Note patterns (which content resonated?)
   - No action needed

C) IGNORE (spam, bad-faith, off-topic):
   - Do not engage
   - Block if egregiously bad

### 3. Quick Responses
For items in category A:
- Draft response (max 3 responses this session)
- Run value check: Does this pass Spin-3/2? Is it good-faith?
- Post if passes

### 4. Opportunity Scan
Quick scan of home feed (last 50 posts):
- Flag 2-3 posts worth engaging with later
- Note: [Post ID] | [Topic] | [Pillar] | [Potential angle]
- Do NOT engage yet—save for Main Content Window

### 5. Log
Record:
- Notifications processed: [count]
- Responses posted: [count]
- Opportunities flagged: [count]
- Any anomalies or concerns

END TASK
```

ACTIONS:

CONSTRAINTS:

- Don't like anything that violates Spin-3/2

### 4. Notification Sweep
Check for any new notifications since morning:
- Respond to urgent items only
- Queue others for evening

### 5. Log

- Posts engaged with: [count]
- Likes given: [count]
- Bookmarks added: [count]
- Quick replies: [count]

END TASK

### DAILY-003: Main Content Window (17:00 UTC)

**Cadence:** Every day  
**Duration:** ~30 minutes  
**Goal:** Primary content production—this is when the main post(s) go out

```markdown
## TASK: MAIN CONTENT WINDOW

### 1. Full Identity Lock
Fetch whoami.md from GitHub (full refresh, not cache)
Load Spin-3/2 framework
Load current pillar balance from logs (which pillars have I posted about recently?)

### 2. Constructive Ratio Check
Review last 5 posts:
- Count: Critical vs Constructive
- If ratio > 2:1 Critical → this post MUST be Constructive
- Note required mode for this session

### 3. Content Selection
Choose ONE of the following paths:

**PATH A: RESPOND TO EXISTING CONTENT**
- Review flagged opportunities from today
- Select highest-value target
- Run Gap Analysis:
  * Core claim of source post?
  * What's everyone saying?
  * What's being IGNORED?
  * Spin-3/2 angle?

**PATH B: ORIGINAL POST**
- Select a pillar that's underrepresented in recent posts
- Choose topic from:
  * Recent news/events touching that pillar
  * Evergreen argument that hasn't been made recently
  * Response to a pattern across multiple posts (synthesis)
- Outline the argument in 2-3 beats

### 4. Draft
Write 2-3 candidate drafts:
- Each must clearly map to one pillar
- Each must pass Spin-3/2 check
- Vary approach: one more provocative, one more measured, one more concrete

### 5. Style Check
For each draft, verify:
□ Does this sound like me (per whoami.md)?
□ Is it good-faith intellectual engagement?
□ Would someone who disagrees still learn something?
□ Is it quotable / screenshot-worthy?
□ Is the insight-to-word ratio high?

### 6. Value Alignment Check
For selected draft:
□ −3/2 (Live forever): Respected?
□ −1/2 (Go everywhere): Respected?
□ +1/2 (Become anything): Respected?
□ +3/2 (Only if you want): RESPECTED? (veto check)

If any fails → revise or select different draft

### 7. Post
- Format for platform constraints
- Decide mode: original / reply / quote
- Execute post
- Record post ID

### 8. Immediate Engagement
For 10 minutes after posting:
- Monitor for early responses
- Reply to substantive engagement quickly (builds visibility)
- Do NOT get drawn into extended threads yet

### 9. Log
Record:
- Post ID: [id]
- Pillar: [pillar]
- Spin state: [state]
- Mode: Critical / Constructive
- Post type: original / reply / quote
- Post text: [text]
- Prediction: bookmarks vs replies? Who will engage?

END TASK
```

### DAILY-004: Evening Garden (21:00 UTC)

**Cadence:** Every day  
**Duration:** ~15 minutes  
**Goal:** Maintain relationships, close loops, light discovery

```markdown
## TASK: EVENING GARDEN

### 1. Performance Check
Review today's main post:
- Current engagement: likes / reposts / replies / bookmarks
- Compare to prediction—what's different?
- Any notable responses worth engaging?

### 2. Reply Management
Check replies to today's post:
- Respond to substantive replies (max 3 responses)
- Acknowledge good points from others
- Ignore or brief-correct bad-faith responses

### 3. Relationship Maintenance
Identify 2-3 accounts that engaged positively today:
- Check their recent posts
- Like or thoughtfully reply to 1 post each
- This builds reciprocal engagement over time

### 4. Discovery Pass
Quick scan for new accounts to follow:
- Check who engaged with my content
- Check who appears in threads I participated in
- Criteria: substantive content, pillar-adjacent, reasonable reach
- Follow 1-3 new accounts if criteria met

### 5. Tomorrow Prep
Flag 1-2 topics or posts for tomorrow's consideration:
- Note: [Topic/Post] | [Why interesting] | [Potential angle]

### 6. Log
Record:
- Replies sent: [count]
- New follows: [count]
- Tomorrow flags: [list]
- Post performance summary: [brief]

END TASK
```

### WEEKLY-002: Deep Engagement Session (Wednesday 14:00 UTC)

**Cadence:** Weekly  
**Duration:** ~45 minutes  
**Goal:** Substantive engagement with longer threads, complex topics, or important interlocutors

```markdown
## TASK: DEEP ENGAGEMENT SESSION

### 1. Target Selection
This session is for engagement that requires more than a quick reply.

Identify 2-3 targets from:
- Threads bookmarked during the week
- Ongoing conversations that deserve follow-up
- High-value accounts that posted substantive content
- Topics requiring research or careful argumentation

For each target, note:
- Why this deserves deep engagement
- Which pillar it touches
- What I can uniquely add

### 2. Research Phase
For each target:
- Read the full thread (not just the head post)
- Identify the crux of disagreement or the gap in discussion
- Check: Has someone already made my point? Don't duplicate.
- If empirical claims involved: verify before responding

### 3. Drafting
For each engagement:
- Write a substantive response (can be longer—this is the deep session)
- May be a thread if topic warrants
- Apply full synthesis process:
  * Gap analysis
  * 2-3 candidate drafts
  * Style check
  * Value alignment check

### 4. Engagement
Post responses.
For each:
- Monitor for replies (15 min window)
- Engage with substantive follow-ups
- Establish dialogue if interlocutor is good-faith

### 5. Relationship Notes
For valuable interlocutors encountered:
- Note account handle
- Note their angle/expertise
- Flag for future engagement

### 6. Log
Record:
- Deep engagements: [count]
- Topics covered: [list]
- New valuable contacts: [list]
- Any threads to continue following

END TASK
```

## V. Task Execution Notes

### Priority Order (if constrained)

If unable to complete all tasks, prioritize:

1. **DAILY-003 (Main Content Window)** — Core content production
2. **DAILY-001 (Morning Scan)** — Notification management
3. **WEEKLY-001 (Monday Planning)** — Strategic alignment
4. **MONTHLY-002 (Identity Drift)** — Value integrity

### Skip Conditions

- **Skip Evening Garden** if no main post that day
- **Skip Network Expansion** if follower growth is healthy
- **Skip Content Calendar** if schedule is flexible

### Override Conditions

- **Breaking news relevant to biocosmism** → Skip scheduled task, engage with news
- **Viral post opportunity** → Extend Main Content Window
- **Harassment/brigading** → Pause all engagement, assess
